INTRODUCTION
------------

This module adds a "Report a bug" link to the admin toolbar and allows you to
configure where it links to. It assumes that you have an external form already
set up to capture bugs.


REQUIREMENTS
------------

This module requires the core Toolbar module to be enabled.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit
   https://www.drupal.org/node/895232/ for further information.


CONFIGURATION
-------------

 * Set the link to your bug report form, either by going to Reports > Bug
   report settings from the admin toolbar, or by visiting this URL directly:
   /admin/reports/bug-report-link/settings.
